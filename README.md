# WireframeToApp

An AI-powered tool that converts hand-drawn UI sketches into live React + Tailwind code using Claude Vision API.

## Features
- **Canvas Drawing Engine**: Smooth HTML5 canvas drawing board with brush configurations.
- **Claude Vision Pipeline**: Integrated streaming engine parsing layout boxes to custom functional JSX components.
- **Sandpack Preview**: Hot reloading browser previews with custom Tailwind styles, logs, and viewport adjustments.
- **Dashboard Management**: Track projects, version lists, exports, and shares.
